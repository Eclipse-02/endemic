## Deskripsi Aplikasi

Endemic adalah aplikasi katalog ensiklopedia hewan dan tumbuhan endemik Indonesia yang dikembangkan untuk platform Android menggunakan Java. 
Aplikasi ini dirancang untuk memberikan pengalaman edukatif yang interaktif untuk menjelajahi berbagai spesies unik nusantara, 
melihat rincian klasifikasi biologis secara mendalam, serta mengelola koleksi favorit pribadi. Proyek ini mengimplementasikan konsep Material Design 3, 
sinkronisasi data API real-time ke database lokal menggunakan ROOM, serta manajemen tema dinamis yang responsif.

## Fitur Utama

- **Endemic Explorer**: Menampilkan daftar spesies yang dikategorikan menjadi *Animals* dan *Plants* dengan menggunakan RecyclerView.
- **Deep Species Insights**: Menyediakan informasi lengkap spesies mencakup nama latin, famili, genus, deskripsi habitat, asal wilayah, serta status konservasi yang informatif.
- **Search & Filter System**: Fitur pencarian spesies yang cepat dengan dukungan eksplorasi berdasarkan wilayah (Sumatera, Jawa, Kalimantan, Sulawesi, Papua) melalui sistem *chip*.
- **Persistent Favorite System**: Fitur untuk menyimpan spesies pilihan ke dalam daftar favorit yang tersimpan secara permanen di database lokal.
- **Dynamic Themes**: Dukungan mode Gelap (Dark Mode) dan Terang (Light Mode).
- **Responsive UI Layouts**: Antarmuka yang adaptif mengikuti standar Material 3 dengan dukungan *Bottom Navigation* untuk navigasi kategori utama.
- **Advanced Media Handling**: Integrasi Glide dengan konfigurasi custom User-Agent untuk optimalisasi pemuatan gambar.
